## Key Decisions Reflected

### API Design
- ✅ Single `/generate` endpoint (idempotent)
- ✅ Locale via query parameter with ENV default
- ✅ Returns existing report without regeneration

### Authentication
- ✅ Single shared bearer token via `COGENCE_API_TOKEN`

### Atomic Commits
- ✅ Simple file count rule (>10 files configurable)
- ✅ Educational Management Notes when detected
- ✅ Metadata includes counts for debugging
- ✅ No individual surveillance or rankings

### Report Generation
- ✅ English prompts with Persian output instruction
- ✅ Sequential Gitea fetching with 100ms delay
- ✅ Rocket.Chat message attachments with structured fields

### Deferred to Backlog
- ✅ Diff truncation token limits
- ✅ LLM semantic atomic detection
- ✅ Parallel fetching optimization
- ✅ Smart diff fetching
